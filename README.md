# 🎤 Speech Emotion Recognition

## 📌 Overview

This project uses Audio Processing and Deep Learning to recognize human emotions from speech signals.

The system extracts MFCC (Mel Frequency Cepstral Coefficients) features from audio and classifies emotions using a neural network.

---

## 🎯 Objective

- Recognize emotions from speech
- Learn audio feature extraction
- Understand speech-based AI applications

---
